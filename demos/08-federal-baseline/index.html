08 基线 — 联邦宪法

来源：intent-schema-compiler 中的 immutable_boundaries。Breaking Change 必须走联邦委员会审批，旧版本冻结，新版本独立发布。

当前不可变边界 🔒 已锁定

status.critical
系统级故障语义令牌。视觉映射为红色脉冲，LLM 约束为"禁止建议自动修复"。
immutable: true
destructive-action
高危操作意图契约。ai_prohibited 包含"直接执行修复操作""修改告警阈值配置"。
immutable: true
human-ai-boundary
人机边界基线。human_mandatory 包含"是否触发自动修复""升级路径选择"。
immutable: true
synonym-mapping / "严重"
同义词防火墙。当前映射到 status.critical，置信度阈值 0.95。
immutable: false（可收紧）
