DESIGN.md 解决什么

语义丰富性：色彩、排版、交互、动效、语气。人类设计师用自然语言表达创意意图。

Schema-As-Code 解决什么

语义确定性：边界、权限、校验、拦截。机器用形式化协议锁定不可突破红线。

设计师不需要

学 YAML、写 JSON Schema、理解四层推演。只需写 Markdown 设计文档。

工程师不需要

读设计文档、记规范细节、人工走查。约束自动编译到 IDE 和 CI。
